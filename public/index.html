Everdream Runner

Fully configurable application launcher with monitoring capabilities.

Gitlab

Features

Terminal UI

Interactive color interface right in terminal, working on headless operating systems.

Fully Configurable

Paths, arguments and modes are all fully configurable. Config file is a simple human editable json file.

Monitoring

Everything reported on stderr is stored into a file, in case of crash part of stdout is stored too, helping debugging efforts.

Keep Alive

Runner can restart the command in case of crash or just keeps it alive as long as it is running.

Open Source

Runner is free and open source, coded in rust for performance and stability. Developed under GPLv3 licence.

Safe

Runner works offline and does not "ping home" at all. Codebase tries to be as lean as possible to avoid any bugs and lower attack surface.
